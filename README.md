A modern, responsive, and interactive personal portfolio website showcasing my skills, projects, and experience as a web developer.

## ✨ Features

- **Responsive Design** – Optimized for desktop, tablet, and mobile devices
- **Modern UI/UX** – Clean, minimal, and professional design

- **Smooth Animations** – Scroll animations and hover effects

- **Fast Performance** – Optimized for speed and SEO
